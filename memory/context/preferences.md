# Preferences

## Communication
- Concise over verbose
- No filler phrases ("Great question!", "I'd be happy to help!")
- Have opinions, disagree when warranted

## Coding Style
- TypeScript with strict types
- Next.js App Router
- Tailwind CSS
- BJJ belt color progression (emerald → blue → purple → gold)
- TDD: write tests first
- Small, focused commits

## Design
- Professional but warm
- Glassmorphism effects on light backgrounds
- WCAG AA accessibility (5:1 contrast minimum)
- Light mode only (dark mode disabled)
- Inter font
